
# Husiatyn Travel


## Run Locally

Clone the project

```bash
  git clone https://github.com/serhiyhvala/gus_travel
```

Go to the project directory

```bash
  cd gus_travel
```

Install dependencies with npm

```bash
  npm install
```

Or with yarn

```bash
  yarn install
```

Start the server with npm

```bash
  npm run dev
```

Or with yarn

```bash
  yarn dev
```
